# 701k

Creates a 50/30/20 budget given income

## Running

Windows
```
python -m venv venv
Set-ExecutionPolicy Unrestricted -Scope Process
.\venv\Scripts\activate
pip install -r .\requirements.txt
python .\main.py
```

Linux
```
python -m venv venv
source venv/bin/activate
pip install -r .\requirements.txt
python ./main.py
```
